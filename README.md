
# Task 4: Feature Encoding & Scaling

## Dataset
Adult Income Dataset

## Objective
To preprocess data using encoding and scaling techniques for ML readiness.

## Steps Performed
- Identified categorical and numerical features
- Applied Label Encoding to target variable
- Applied One-Hot Encoding to categorical features
- Scaled numerical features using StandardScaler
- Saved processed dataset

## Outcome
The dataset is fully prepared for machine learning models.
